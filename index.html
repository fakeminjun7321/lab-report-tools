보고서 도구 모음

실험 보고서를 쓸 때 자주 필요한 작업을 빠르게 끝내는 도구들입니다. 모두 브라우저 안에서만 동작하고 입력 내용은 서버로 전송되지 않습니다.

글자수 세기

공백 포함/미포함 글자수, 단어수, 줄수, 바이트(UTF-8)를 실시간으로 셉니다. 보고서 분량 제한 맞출 때.

텍스트 → 글자수·바이트

평균·표준편차 계산기

측정값을 붙여넣으면 개수·평균·표준편차·표준오차·범위를 계산합니다. 결과 보고서 데이터 정리에.

데이터 → 평균·SD·SE

유효숫자·오차 계산기

백분율 오차, 유효숫자 반올림, 기본 오차 전파(덧셈·곱셈)를 계산합니다. 측정값과 이론값 비교에.

측정·이론 → % 오차

LaTeX → 한글 수식 변환기

LaTeX 수식을 한글(HWP) 수식으로 바꿔 줍니다. 텍스트 변환과 HWP·HWPX 파일 변환 모두 지원.

LaTeX → 한글 수식

텍스트 정리

여러 공백·빈 줄·줄바꿈을 정리해 깔끔한 평문으로 만듭니다. 붙여넣기 정리에.

텍스트 → 평문 정리

이미지 변환·압축

사진을 JPG·PNG·WebP로 바꾸고 크기를 줄입니다. 보고서에 넣을 사진 용량 줄일 때.

이미지 → 변환·압축

선형회귀·추세선

(x, y) 데이터로 기울기·절편·R²을 구하고 추세선 그래프를 그려 PNG로 받습니다. 물리 실험에.

데이터 → 회귀·추세선

그래프 생성기

데이터를 붙여넣어 산점도·꺾은선·막대 그래프를 그리고 PNG로 받습니다. 보고서에 바로 삽입.

데이터 → 그래프 PNG

PDF 병합

여러 PDF를 원하는 순서로 하나로 합칩니다. 브라우저 안에서만 처리.

여러 PDF → 한 파일

파일 변환기

표(Excel↔CSV↔TSV), 이미지→PDF, PDF→이미지, 이미지 형식을 서로 변환합니다.

표·이미지·PDF 변환
